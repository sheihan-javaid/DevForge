## What is DevForge?

DevForge is a collection of free, fast, and privacy-friendly tools built for developers. Whether you need to format some JSON, decode a JWT, generate a UUID, or test a regex — DevForge has you covered, right in your browser.

No installation. No account. No data sent to any server. Just open the tool and use it.

---
## 🧰 Tools

### 📋 Formatters & Validators
- **JSON Formatter** — Prettify, minify, and validate JSON
- **Regex Tester** — Test regular expressions with live match highlighting
- **Diff Checker** — Compare two blocks of text and highlight differences
- **Cron Parser** — Parse and explain cron expressions in plain English
- **YAML ↔ JSON** — Convert between YAML and JSON formats

### 🔐 Encoders & Decoders
- **Base64 Encoder / Decoder** — Encode or decode Base64 strings and files
- **URL Encoder / Decoder** — Encode or decode URL components and query strings
- **JWT Decoder** — Decode and inspect JWT header, payload, and signature

### ⚙️ Generators
- **UUID Generator** — Generate v4 and v7 UUIDs in bulk
- **Password Generator** — Generate strong passwords with entropy scoring
- **Hash Generator** — Generate MD5, SHA-1, SHA-256, SHA-384, SHA-512 hashes
- **Lorem Ipsum Generator** — Generate placeholder text for designs and mockups
- **Random String Generator** — Generate random strings with custom charset and format

### 🔄 Converters
- **Unix Timestamp Converter** — Convert Unix timestamps to readable dates and back

### 🔤 Text & String
- **String Case Converter** — Convert between camelCase, snake_case, kebab-case, and 13 more

## 🔒 Privacy

DevForge takes your privacy seriously:

- **Nothing is sent to a server** — all processing happens in your browser
- **No account required** — use every tool without signing up
- **No tracking** — no analytics, no fingerprinting, no cookies
- **No ads** (currently) — clean, distraction-free experience
- **Open source** — the code is publicly auditable on GitHub

This means you can safely paste sensitive data like JWT tokens, passwords, API keys, and private text without any risk.
